# Plugin Option: Infection

`infection` is a plugin option for infection-like propagation behavior.

## Example

```ts
infection: {
  stages: [
    {
      color: {
        value: "#22c55e",
      },
      radius: 20,
    },
  ],
}
```

## Notes

- Use clear stage definitions for predictable behavior.
- Test visually with small particle counts first.
