# Opacidad de partículas

`particles.opacity` controla la animación de transparencia y opacidad.

## Ejemplo

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
