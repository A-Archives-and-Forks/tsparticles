# Las partículas se tambalean

`particles.wobble` agrega movimiento oscilante alrededor de la ruta de movimiento.

## Ejemplo

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
