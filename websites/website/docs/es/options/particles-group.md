# Grupo de partículas

`particles.groups` te permite definir variantes de partículas con nombre en la misma escena.

## Ejemplo

```ts
particles: {
  groups: {
    blue: {
      color: {
        value: "#60a5fa",
      },
      number: {
        value: 30,
      },
    },
    pink: {
      color: {
        value: "#f472b6",
      },
      number: {
        value: 20,
      },
    },
  },
}
```
