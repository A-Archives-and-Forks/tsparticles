# Tamaño de partículas

`particles.size` controla el rango de tamaño de partículas y la animación de tamaño.

## Ejemplo

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
