# Particules vacillantes

`particles.wobble` ajoute un mouvement oscillant autour de la trajectoire de déplacement.

## Exemple

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
