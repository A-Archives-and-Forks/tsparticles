# Opacité des particules

`particles.opacity` contrôle l'animation de transparence et d'opacité.

## Exemple

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
