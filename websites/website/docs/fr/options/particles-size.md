# Taille des particules

`particles.size` contrôle la plage de taille des particules et l'animation de taille.

## Exemple

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
