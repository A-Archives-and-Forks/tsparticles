# Orbite des particules

`particles.orbit` fait tourner les particules autour des centres orbitaux.

## Exemple

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable` : active le comportement de l'orbite.
- `radius` : distance orbitale.
- `rotation` : contrôle de l'angle d'orbite.
