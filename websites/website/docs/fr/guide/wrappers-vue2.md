# Emballage : @tsparticles/vue2

Wrapper officiel Vue 2 pour `tsParticles`.

## Installer

```bash
pnpm add @tsparticles/vue2 @tsparticles/engine @tsparticles/slim
```

## Flux de configuration rapide

1. Installez le package wrapper + moteur + chargeur.
2. Enregistrez le plugin une fois dans le bootstrap de l'application Vue.
3. Rendre le composant Vue avec l'objet options.
