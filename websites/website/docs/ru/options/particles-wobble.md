# Колебание частиц

`particles.wobble` добавляет колебательное движение по траектории перемещения.

## Пример

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
