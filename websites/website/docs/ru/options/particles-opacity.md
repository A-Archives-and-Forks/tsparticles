# Непрозрачность частиц

`particles.opacity` управляет анимацией прозрачности и непрозрачности.

## Пример

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
