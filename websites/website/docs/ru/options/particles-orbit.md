# Орбита частиц

`particles.orbit` заставляет частицы вращаться вокруг центров орбит.

## Пример

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: активирует поведение орбиты.
- `radius`: расстояние по орбите.
- `rotation`: контроль угла орбиты.
