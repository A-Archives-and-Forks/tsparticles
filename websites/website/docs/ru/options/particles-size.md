# Размер частиц

`particles.size` управляет диапазоном размеров частиц и анимацией размера.

## Пример

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
