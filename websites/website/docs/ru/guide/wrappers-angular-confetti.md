# Обертка: angular-confetti

Официальная оболочка Angular вокруг `@tsparticles/confetti`.

## Установить

```bash
pnpm add angular-confetti @tsparticles/confetti
```

## Быстрая настройка

1. Установите обертку + `@tsparticles/confetti`.
2. Выполните инициализацию один раз при запуске приложения Angular.
3. Запускайте действия конфетти из компонентов/сервисов.
