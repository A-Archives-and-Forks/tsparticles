# Modelos e recursos

Os modelos são mantidos no monorepo principal na pasta `templates`.

## Modelos oficiais

- Pasta de modelos: <https://github.com/tsparticles/tsparticles/tree/main/templates>
- Inclui configurações iniciais para JavaScript vanilla e vários frameworks.

## Quando usar modelos

- Você deseja uma linha de base pronta com a estrutura do projeto já instalada.
- Você precisa de exemplos específicos da estrutura antes de escrever wrappers personalizados.
- Você deseja comparar opções de pacotes (`basic`, `slim`, `all`) rapidamente.

## Fluxo de contribuição

Se você criar um design reutilizável ou uma receita de integração:

1. Bifurque o monorepo principal.
2. Adicione seu modelo com instruções claras de configuração e execução.
3. Abra uma solicitação pull e inclua capturas de tela/gif e versões do pacote.
