# Tamanho das Partículas

`particles.size` controla a faixa de tamanho de partícula e a animação de tamanho.

## Exemplo

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
