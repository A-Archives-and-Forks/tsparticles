# Partículas oscilam

`particles.wobble` adiciona movimento oscilante ao redor do caminho do movimento.

## Exemplo

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
