# Partículas Brilham

`particles.twinkle` adiciona efeitos piscantes semelhantes a brilhos.

## Exemplo

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## Orientação prática

- Comece com `frequency` baixo para efeitos sutis.
- Combine com fundos escuros para melhor visibilidade.
