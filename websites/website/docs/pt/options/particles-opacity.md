# Opacidade de partículas

`particles.opacity` controla a animação de transparência e opacidade.

## Exemplo

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
