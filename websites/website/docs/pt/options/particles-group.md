# Grupo de Partículas

`particles.groups` permite definir variantes de partículas nomeadas na mesma cena.

## Exemplo

```ts
particles: {
  groups: {
    blue: {
      color: {
        value: "#60a5fa",
      },
      number: {
        value: 30,
      },
    },
    pink: {
      color: {
        value: "#f472b6",
      },
      number: {
        value: 20,
      },
    },
  },
}
```
