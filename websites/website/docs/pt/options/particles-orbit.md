# Órbita de Partículas

`particles.orbit` faz as partículas girarem em torno dos centros das órbitas.

## Exemplo

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: ativa o comportamento da órbita.
- `radius`: distância da órbita.
- `rotation`: controle do ângulo da órbita.
