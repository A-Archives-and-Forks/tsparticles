# Particles Roll

`particles.roll` applies rolling behavior and can create depth-like visual motion.

## Example

```ts
particles: {
  roll: {
    enable: true,
    speed: 12,
    darken: {
      enable: true,
      value: 20,
    },
  },
}
```

## Practical guidance

- Use with moderate speed first.
- Combine carefully with `tilt` and `rotate` for readability.
