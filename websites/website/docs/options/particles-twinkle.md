# Particles Twinkle

`particles.twinkle` adds sparkle-like blinking effects.

## Example

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## Practical guidance

- Start with low `frequency` for subtle effects.
- Combine with dark backgrounds for better visibility.
