# Particles Orbit

`particles.orbit` makes particles revolve around orbit centers.

## Example

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: activates orbit behavior.
- `radius`: orbit distance.
- `rotation`: orbit angle control.
