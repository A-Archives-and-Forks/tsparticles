# Particles Size

`particles.size` controls particle size range and size animation.

## Example

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
