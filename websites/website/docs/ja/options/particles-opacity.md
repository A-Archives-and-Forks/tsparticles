# パーティクルの不透明度

`particles.opacity` は、透明度と不透明度のアニメーションを制御します。

## 例

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
