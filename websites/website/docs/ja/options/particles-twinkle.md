# パーティクルトゥインクル

`particles.twinkle` は、キラキラしたような点滅効果を追加します。

## 例

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## 実践的な指導

- 微妙な効果を得るには、低い `frequency` から始めます。
- 暗い背景と組み合わせると視認性が向上します。
