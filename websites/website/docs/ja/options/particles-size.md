# 粒子サイズ

`particles.size` は、パーティクル サイズの範囲とサイズのアニメーションを制御します。

## 例

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
