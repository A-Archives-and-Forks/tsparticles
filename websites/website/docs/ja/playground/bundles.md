# プレイグラウンドバンドル

このページを使用して、次のパッケージの焦点を当てた API をテストします。

- `@tsparticles/confetti`
- `@tsparticles/fireworks`
- `@tsparticles/particles`

各セクションには独自のキャンバス、JSON エディター、およびコントロールがあります。

デモはローカルのキャンバス要素 (`*.create(...)`) にスコープされているため、ページ全体を引き継ぐことはありません。
