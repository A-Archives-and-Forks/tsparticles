# ショーケース

ショーケースのコンテンツがメインのドキュメント フローに統合されたため、例が複数の小さなページに分割されなくなりました。

## 代わりにこれらのセクションを使用してください

- インタラクティブなデモとレシピ: [`/demos/`](/ja/demos/)
- ライブ編集可能なプレイグラウンド: [`/playground/`](/ja/playground/)
- ラッパー固有の統合ドキュメント: [`/guide/wrappers`](/ja/guide/wrappers)
