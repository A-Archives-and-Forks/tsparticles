# 粒子闪烁

`particles.twinkle` 添加类似闪光的闪烁效果。

## 示例

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## 实用指导

- 从低 `frequency` 开始以获得微妙的效果。
- 与深色背景结合以获得更好的可见性。
