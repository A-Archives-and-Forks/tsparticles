# 粒子滚动

`particles.roll` 应用滚动行为并可以创建类似深度的视觉运动。

## 示例

```ts
particles: {
  roll: {
    enable: true,
    speed: 12,
    darken: {
      enable: true,
      value: 20,
    },
  },
}
```

## 实用指导

- 首先以中等速度使用。
- 与 `tilt` 和 `rotate` 仔细结合以提高可读性。
