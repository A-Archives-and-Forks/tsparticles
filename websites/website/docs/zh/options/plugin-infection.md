# 插件选项：感染

`infection` 是用于类似感染传播行为的插件选项。

## 示例

```ts
infection: {
  stages: [
    {
      color: {
        value: "#22c55e",
      },
      radius: 20,
    },
  ],
}
```

## 注释

- 使用清晰的阶段定义来实现可预测的行为。
- 首先用小颗粒计数进行目视测试。
