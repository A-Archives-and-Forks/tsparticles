# 粒子轨道

`particles.orbit` 使粒子绕轨道中心旋转。

## 示例

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`：激活轨道行为。
- `radius`：轨道距离。
- `rotation`：轨道角度控制。
