# 颗粒大小

`particles.size` 控制粒子尺寸范围和尺寸动画。

## 示例

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
