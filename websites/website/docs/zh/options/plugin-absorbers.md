# 插件选项：吸收器

`absorbers` 是一个插件选项，需要在您的捆绑包/设置中提供吸收器功能。

## 示例

```ts
absorbers: {
  position: {
    x: 50,
    y: 50,
  },
  size: {
    value: 20,
  },
}
```

## 注释

- 在使用吸收器选项之前验证插件的可用性。
- 从一个吸收器实例开始，然后逐渐扩展。
