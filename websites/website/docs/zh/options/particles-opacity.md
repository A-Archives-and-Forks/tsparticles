# 粒子不透明度

`particles.opacity` 控制透明度和不透明度动画。

## 示例

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
