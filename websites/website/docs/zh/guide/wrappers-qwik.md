# 包装：@tsparticles/qwik

`tsParticles` 的官方 Qwik 包装器。

## 安装

```bash
pnpm add @tsparticles/qwik @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在Qwik启动路径中初始化加载程序。3. 使用序列化选项渲染包装器组件。
