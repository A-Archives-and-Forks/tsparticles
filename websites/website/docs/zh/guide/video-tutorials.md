## 推荐的学习顺序

1. 从 [`/guide/getting-started`](/zh/guide/getting-started) 开始。
2. 复制您堆栈中的一个特色样本。
3. 移至 [`/options/`](/zh/options/) 以自定义行为。
4. 使用 [`/guide/plugins-customization`](/zh/guide/plugins-customization) 实现自定义功能。
