# 包装：@tsparticles/inferno

`tsParticles` 的官方 Inferno 包装器。

## 安装

```bash
pnpm add @tsparticles/inferno @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1. 安装wrapper+`@tsparticles/engine`+loader包。
2. 在应用程序初始化中加载功能一次。
3. 使用键入的选项渲染 Inferno 组件。
