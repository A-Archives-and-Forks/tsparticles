# 包装：@tsparticles/ember

`tsParticles` 的官方 Ember 包装器。

## 安装

```bash
pnpm add @tsparticles/ember @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在应用程序引导程序中初始化引擎功能。3. 使用您的选项安装 Ember 组件。
