# 包装：@tsparticles/nuxt4

`tsParticles` 的官方 Nuxt 4 模块包装器。

## 安装

```bash
pnpm add @tsparticles/nuxt4 @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装模块+引擎+加载包。2.注册Nuxt模块并在客户端保留canvas渲染。3. 从应用程序配置/组件传递选项。
