# 包装：@tsparticles/svelte

`tsParticles` 的官方 Svelte 包装器。

## 安装

```bash
pnpm add @tsparticles/svelte @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 初始化引擎功能一次。3. 使用 props 中的选项渲染 Svelte 组件。
