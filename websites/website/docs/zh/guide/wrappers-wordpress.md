# 包装：@tsparticles/wordpress

`tsParticles` 的官方 WordPress 插件包。

## 安装

```bash
pnpm add @tsparticles/wordpress
```

## 注释

- 需要完整安装 WordPress。
- 未映射到此 monorepo 中的独立 `demo/*` 文件夹。

## 快速设置流程

1. 在 WordPress 插件/主题工作流程中安装包。2.通过WordPress钩子注册/排队脚本。
2. 从插件设置或模板传递选项 JSON。
