# 包装器：角度烟花

围绕 `@tsparticles/fireworks` 的官方 Angular 包装器。

## 安装

```bash
pnpm add angular-fireworks @tsparticles/fireworks
```

## 快速设置流程

1. 安装包装器 + `@tsparticles/fireworks`。
2. 在 Angular 应用引导程序中初始化一次。
3. 从 UI 处理程序触发烟花操作。
