# 包装：@tsparticles/solid

`tsParticles` 的官方实体包装。

## 安装

```bash
pnpm add @tsparticles/solid @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在启动时加载一次选定的功能集。3. 使用选项道具渲染 Solid 组件。
