# 包装：@tsparticles/preact

`tsParticles` 的官方 Preact 包装器。

## 安装

```bash
pnpm add @tsparticles/preact @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在应用程序根目录初始化引擎一次。3. 使用选项对象渲染 Preact 组件。
