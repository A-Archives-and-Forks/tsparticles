# 包装：@tsparticles/riot

`tsParticles` 的官方防暴包装。

## 安装

```bash
pnpm add @tsparticles/riot @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在 Riot 应用程序设置中注册/加载功能一次。3. 使用选项渲染 Riot 包装器。
