# 包装：@tsparticles/nuxt3

`tsParticles` 的官方 Nuxt 3 模块包装器。

## 安装

```bash
pnpm add @tsparticles/nuxt3 @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1. 安装模块+引擎+选定的加载器。
2. 向客户端渲染注册模块/插件。
3. 在组件或可组合设置中配置选项。
