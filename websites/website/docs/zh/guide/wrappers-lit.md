# 包装：@tsparticles/lit

`tsParticles` 的官方 Lit 包装。

## 安装

```bash
pnpm add @tsparticles/lit @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在应用程序启动期间注册/加载功能。3. 使用 Lit 组件并将选项作为 props 传递。
