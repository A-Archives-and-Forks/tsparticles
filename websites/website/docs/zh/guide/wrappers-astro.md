# 包装：@tsparticles/astro

`tsParticles` 的官方 Astro 包装器。

## 安装

```bash
pnpm add @tsparticles/astro @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1. 安装包装器+引擎+选择的加载器（建议`@tsparticles/slim`）。
2. 在客户端的 Astro 组件/布局中注册粒子。
3. 将选项对象或 JSON 传递给包装器组件。
