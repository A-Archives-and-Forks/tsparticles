# 包装：@tsparticles/vue3

`tsParticles` 的官方 Vue 3 包装器。

## 安装

```bash
pnpm add @tsparticles/vue3 @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在应用程序启动时注册一次插件。3. 使用选项和处理程序渲染 Vue 组件。
