# 模板和资源

模板维护在主 monorepo 的 `templates` 文件夹下。

## 官方模板

- 模板文件夹：<https://github.com/tsparticles/tsparticles/tree/main/templates>
- 包括原生 JavaScript 和多个框架的入门设置。

## 何时使用模板

- 您需要一个已连接项目结构的现成基线。
- 在编写自定义包装器之前，您需要特定于框架的示例。
- 您想要快速比较捆绑选项（`basic`、`slim`、`full`、`all`）。

## 贡献流程

如果您构建可重用的设计或集成方案：

1. 分叉主要的 monorepo。
2. 添加具有清晰设置和运行说明的模板。
3. 打开拉取请求并包含屏幕截图/gif 和包版本。
