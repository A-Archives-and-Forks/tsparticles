# 包装：@tsparticles/vue2

`tsParticles` 的官方 Vue 2 包装器。

## 安装

```bash
pnpm add @tsparticles/vue2 @tsparticles/engine @tsparticles/slim
```

## 快速设置流程

1.安装wrapper+engine+loader包。2. 在Vue应用程序引导程序中注册一次插件。3. 使用选项对象渲染 Vue 组件。
