# Dimensione delle particelle

`particles.size` controlla l'intervallo di dimensioni delle particelle e l'animazione delle dimensioni.

## Esempio

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
