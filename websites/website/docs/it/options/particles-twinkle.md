# Le particelle brillano

`particles.twinkle` aggiunge effetti lampeggianti simili a scintillii.

## Esempio

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## Guida pratica

- Inizia con `frequency` basso per effetti delicati.
- Combinalo con sfondi scuri per una migliore visibilità.
