# Opacità delle particelle

`particles.opacity` controlla la trasparenza e l'animazione dell'opacità.

## Esempio

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
