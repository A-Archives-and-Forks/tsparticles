# Orbita di particelle

`particles.orbit` fa ruotare le particelle attorno ai centri dell'orbita.

## Esempio

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: attiva il comportamento orbitale.
- `radius`: distanza dell'orbita.
- `rotation`: controllo dell'angolo dell'orbita.
