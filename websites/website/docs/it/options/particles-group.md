# Gruppo di particelle

`particles.groups` ti consente di definire varianti di particelle con nome nella stessa scena.

## Esempio

```ts
particles: {
  groups: {
    blue: {
      color: {
        value: "#60a5fa",
      },
      number: {
        value: 30,
      },
    },
    pink: {
      color: {
        value: "#f472b6",
      },
      number: {
        value: 20,
      },
    },
  },
}
```
