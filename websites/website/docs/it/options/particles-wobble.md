# Le particelle oscillano

`particles.wobble` aggiunge movimento oscillatorio attorno al percorso di spostamento.

## Esempio

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
