# कणों का आकार

`particles.size` कण आकार सीमा और आकार एनीमेशन को नियंत्रित करता है।

## उदाहरण

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
