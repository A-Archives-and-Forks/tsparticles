# कण कक्षा

`particles.orbit` कणों को कक्षा केंद्रों के चारों ओर चक्कर लगाता है।

## उदाहरण

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: कक्षा व्यवहार को सक्रिय करता है।
- `radius`: कक्षा दूरी।
- `rotation`: कक्षा कोण नियंत्रण।
