# कण टिमटिमाते हैं

`particles.twinkle` चमक-जैसा चमकीला प्रभाव जोड़ता है।

## उदाहरण

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## व्यावहारिक मार्गदर्शन

- सूक्ष्म प्रभावों के लिए कम `frequency` से प्रारंभ करें।
- बेहतर दृश्यता के लिए गहरे रंग की पृष्ठभूमि के साथ संयोजन करें।
