# कण अपारदर्शिता

`particles.opacity` पारदर्शिता और अपारदर्शिता एनीमेशन को नियंत्रित करता है।

## उदाहरण

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
