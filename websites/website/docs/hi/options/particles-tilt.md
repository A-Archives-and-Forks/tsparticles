# कण झुकाव

`particles.tilt` झुकाव कोण और झुकाव एनीमेशन को नियंत्रित करता है।

## उदाहरण

```ts
particles: {
  tilt: {
    enable: true,
    direction: "clockwise",
    value: {
      min: 0,
      max: 360,
    },
    animation: {
      enable: true,
      speed: 15,
      sync: false,
    },
  },
}
```
