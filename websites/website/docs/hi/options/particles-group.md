# कण समूह

`particles.groups` आपको एक ही दृश्य में नामित कण वेरिएंट को परिभाषित करने देता है।

## उदाहरण

```ts
particles: {
  groups: {
    blue: {
      color: {
        value: "#60a5fa",
      },
      number: {
        value: 30,
      },
    },
    pink: {
      color: {
        value: "#f472b6",
      },
      number: {
        value: 20,
      },
    },
  },
}
```
