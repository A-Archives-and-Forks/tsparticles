# प्लगइन विकल्प: एमिटर

`emitters` कणों को गतिशील रूप से उत्पन्न करता है और प्लगइन-संचालित होता है।

## उदाहरण

```ts
emitters: {
  position: {
    x: 50,
    y: 50,
  },
  rate: {
    quantity: 5,
    delay: 0.2,
  },
}
```

## नोट्स

- विस्फोट प्रभाव और गतिशील कण निर्माण के लिए बढ़िया।
- प्रदर्शन में बढ़ोतरी से बचने के लिए उत्सर्जन दरें संतुलित रखें।
