# कण रोल

`particles.roll` रोलिंग व्यवहार लागू करता है और गहराई जैसी दृश्य गति बना सकता है।

## उदाहरण

```ts
particles: {
  roll: {
    enable: true,
    speed: 12,
    darken: {
      enable: true,
      value: 20,
    },
  },
}
```

## व्यावहारिक मार्गदर्शन

- पहले मध्यम गति से प्रयोग करें।
- पठनीयता के लिए `tilt` और `rotate` को सावधानीपूर्वक संयोजित करें।
