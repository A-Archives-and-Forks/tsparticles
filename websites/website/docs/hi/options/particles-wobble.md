# कण डगमगाते हैं

`particles.wobble` चाल पथ के चारों ओर दोलन गति जोड़ता है।

## उदाहरण

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
