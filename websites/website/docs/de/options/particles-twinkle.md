# Teilchen funkeln

`particles.twinkle` fügt funkelnde Blinkeffekte hinzu.

## Beispiel

```ts
particles: {
  twinkle: {
    particles: {
      enable: true,
      frequency: 0.03,
      color: {
        value: "#ffffff",
      },
      opacity: 1,
    },
  },
}
```

## Praktische Anleitung

- Beginnen Sie mit einem niedrigen `frequency` für subtile Effekte.
- Für bessere Sichtbarkeit mit dunklen Hintergründen kombinieren.
