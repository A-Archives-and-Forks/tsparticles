# Partikel wackeln

`particles.wobble` fügt eine oszillierende Bewegung um den Bewegungspfad hinzu.

## Beispiel

```ts
particles: {
  wobble: {
    enable: true,
    distance: 10,
    speed: {
      min: -7,
      max: 7,
    },
  },
}
```
