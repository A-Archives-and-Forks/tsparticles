# Partikelgruppe

Mit `particles.groups` können Sie benannte Partikelvarianten in derselben Szene definieren.

## Beispiel

```ts
particles: {
  groups: {
    blue: {
      color: {
        value: "#60a5fa",
      },
      number: {
        value: 30,
      },
    },
    pink: {
      color: {
        value: "#f472b6",
      },
      number: {
        value: 20,
      },
    },
  },
}
```
