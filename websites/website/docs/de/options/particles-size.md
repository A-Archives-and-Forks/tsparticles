# Partikelgröße

`particles.size` steuert den Partikelgrößenbereich und die Größenanimation.

## Beispiel

```ts
particles: {
  size: {
    value: {
      min: 1,
      max: 5,
    },
    animation: {
      enable: true,
      speed: 2,
      sync: false,
    },
  },
}
```
