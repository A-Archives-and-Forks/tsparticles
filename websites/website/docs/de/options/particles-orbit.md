# Teilchenbahn

`particles.orbit` lässt Teilchen um Umlaufbahnzentren kreisen.

## Beispiel

```ts
particles: {
  orbit: {
    enable: true,
    radius: 30,
    rotation: {
      value: 45,
    },
  },
}
```

- `enable`: Aktiviert das Orbitverhalten.
- `radius`: Umlaufbahnentfernung.
- `rotation`: Steuerung des Umlaufwinkels.
