# Partikelopazität

`particles.opacity` steuert die Transparenz- und Deckkraftanimation.

## Beispiel

```ts
particles: {
  opacity: {
    value: {
      min: 0.2,
      max: 0.8,
    },
    animation: {
      enable: true,
      speed: 1,
      sync: false,
    },
  },
}
```
