## Empfohlene Lernreihenfolge

1. Beginnen Sie mit [`/guide/getting-started`](/de/guide/getting-started).
2. Reproduzieren Sie ein ausgewähltes Beispiel in Ihrem Stapel.
3. Gehen Sie zu [`/options/`](/de/options/), um das Verhalten anzupassen.
4. Verwenden Sie [`/guide/plugins-customization`](/de/guide/plugins-customization) für benutzerdefinierte Funktionen.
