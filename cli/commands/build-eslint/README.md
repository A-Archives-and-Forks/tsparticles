# tsParticles CLI

## Installation

### NPM

```bash
npm install -g @tsparticles/cli-build
```

### Yarn

```bash
yarn global add @tsparticles/cli-build
```

### PNPM

```bash
pnpm global add @tsparticles/cli-build
```

## Usage

### Help

```bash
npx @tsparticles/cli-build --help
```

or

```bash
tsparticles-build --help
```

### Build

```bash
npx @tsparticles/cli-build
```

or

```bash
tsparticles-build
```
